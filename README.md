# vuejs_moderne

E-commerce frontend in vuejs

## Project Setup

Installation of thr project

```sh
git clone https://github.com/Alphonse243/vuejs_moderne_e-commerce.git

cd vuejs_moderne_e-commerce.
```
Installation of dependances

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
